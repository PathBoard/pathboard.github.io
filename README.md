# PathBoard - Website

## Introduction


### Running protocol

1. Create a new MkDocs project
```bash
mkdocs new .
```

2. Open a new testing (ASGI) server to check the update on web app 
```bash
mkdocs serve
```

3. Deploy on production server
```bash
mkdocs gh-deploy
```
